# Agent Skills Collection

A collection of Claude Skills for various development tasks.

## Available Skills

### jtl-nova-template

JTL-Shop 5 NOVA template customization skill for creating Child-Templates, SCSS theming, Smarty modifications, and professional parametric product catalogs (DigiKey-style).

**Features:**
- Child-Template setup and configuration
- SCSS/CSS variable customization
- Smarty template block inheritance
- Parametric product catalog with gallery/list/table views
- Product filtering and comparison
- CSV/Excel export functionality

**Usage:** Copy the `jtl-nova-template` folder to your Claude Skills directory or upload the `.skill` file.

## Skill Structure

Each skill follows the Claude Skills standard format:

```
skill-name/
├── SKILL.md           # Core instructions (required)
├── references/        # Detailed documentation (loaded on demand)
├── assets/           # Templates, scripts, resources
└── scripts/          # Executable scripts (if any)
```
