# Agent Skills

A collection of reusable skills for AI agents.

## Overview

This repository contains a curated set of agent skills that can be used to extend and enhance AI agent capabilities. Each skill is designed to be modular, reusable, and easy to integrate.

## Structure

```
Agent-Skills/
├── README.md
└── skills/
    └── <skill-name>/
        ├── README.md       # Skill documentation
        ├── prompt.md       # Skill prompt/instructions
        └── examples/       # Usage examples
```

## Usage

Each skill contains its own documentation with specific usage instructions. Browse the `skills/` directory to explore available skills.

## Contributing

To add a new skill:

1. Create a new directory under `skills/` with your skill name
2. Include a `README.md` describing the skill's purpose and usage
3. Add the skill prompt/instructions in `prompt.md`
4. Provide examples where applicable
